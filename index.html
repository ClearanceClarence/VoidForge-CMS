Features

Everything you need, nothing you don't

Forge CMS packs powerful features into a clean, focused package. No overwhelming options — just the tools you actually need.

Custom Post Types

Create products, portfolios, testimonials, or any content type. Visual builder with automatic admin menus and frontend routing.

Custom Fields

Add text, numbers, dates, colors, images, and more to any post type. 14 field types with an intuitive builder interface.

Theme System

Build themes with pure PHP templates. No proprietary languages — just familiar code with powerful template tags.

Media Library

Drag-and-drop uploads with folder organization. Automatic thumbnails, alt text, and image optimization built in.

User Roles

Role-based access control with Admin, Editor, and Author roles. Secure authentication with bcrypt password hashing.

Auto Updates

One-click updates with automatic backups and database migrations. Never worry about manual upgrades again.

Built for developers who ship fast

Forge CMS gets out of your way. Simple template functions, an intuitive custom fields API, and zero configuration needed. Your code, your rules.

  • Function-based API for custom fields
  • Post-type specific templates
  • WordPress-style hooks & filters
  • Zero external dependencies
